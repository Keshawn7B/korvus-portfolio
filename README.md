# Personal Portfolio Workspace

This is the working folder for building a public portfolio around coding and creative projects.

## Purpose

Turn scattered work into a clear showcase:

- project pages
- screenshots and demos
- GitHub links
- writeups
- skills learned
- deployment links
- future project ideas

## Folder layout

```text
content/      Project writeups and portfolio copy
assets/       Logos, design assets, images
screenshots/  Screenshots for project pages
demos/        Demo notes, links, recordings, builds
notes/        Planning notes and raw ideas
```

## Current operating rule

Korvus can work freely inside this folder and should keep useful changes in git.
